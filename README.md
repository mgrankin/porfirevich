# porfirevich

UI for [ru_transformers](https://github.com/mgrankin/ru_transformers)
[LIVE](https://text.skynet.center/)

## Project setup

```bash
npm install
```

## Production

```bash
npm run build
npm start
```

## Development

```bash
npm run express
npm run serve
```

### Run your tests

```bash
npm run test
```

### Lints and fixes files

npm run lint
